# Claw-Prime

**Claw-Prime** is a high-leverage autonomous task engine for revenue generation and context engineering.

## Purpose

This repository houses the core autonomous skills, research documents, and task workflows for OpenClaw. It serves as the primary engine for automating workflows, curating intelligence, and driving revenue-generating agent tasks.

## Contents

- `skills/`: Modular OpenClaw capabilities (e.g., AI article curator, blog generator, image gen).
- `researches/`: In-depth analysis and context documents (e.g., agent-harness, agent-skills).

## Note

Personal configuration files (like `MEMORY.md`, `TOOLS.md`, etc.) and separate brand assets have been untracked or moved to maintain this repository's focus solely on the autonomous engine.
